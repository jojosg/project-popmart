## 📝 프로젝트 소개

중국의 디자이너 토이 브랜드 POP MART의 제품 소개 및 브랜드 경험을 제공하는 반응형 웹사이트


## ✨ 주요 기능

- **회원가입**: 로그인 페이지에서 회원가입 창으로 이동하여 휴대폰 인증이 가능하고 회원가입이 가능합니다

- **장바구니**: BEST 및 COLLECTION 탭에서 마음에드는 상품을 장바구니에 담고 장바구니 탭에서 확인이 가능합니다

- **주문**: 장바구니탭에서 확인한 제품들중 마음에 드는 상품에 주문하기 버튼을 누르면 주문이 완료됩니다

- **검색 기능**: 우상단에 돋보기 탭을 누르면 텍스트로 제품 검색이 가능하고 해당 제품 페이지로 넘어갑니다
